# Intalacion de programas

## Node.Js

*A travez de "NPM" y "n"*

```
# Update Packages
sudo apt-get update

# Upgrade Packages
sudo apt-get upgrade
sudo apt-get dist-upgrade

# Instalar npm y node (V 14.17.1)
sudo apt install npm -y
sudo npm cache clean -f
sudo npm install -g n
sudo n 14.17.1
```
